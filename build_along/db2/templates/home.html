Welcome

It is a mini-project. It builds a website by utilising a local database, it includes the ForeginKey and relationship. Used technologies are the following

  • Flask
  • SQLalchemy
  • Bootstrap

Things I learned

  • Do the form.submit(class_='foo') later, make sure everything works before caring the css/appearance, otherwise something strange will happen.
  • TYPE CAREFULLY, there are a lot of typo which make a lot of bugs.
  • whenever there is a structural change, for example, ForeginKey relationship, migrate the and upgrate the new version before HTML rendering.
  • Must initiate the __tablename__ in the Class foo(SQLAlchemy.Model), the SQLAlchemy.ForeignKey() requires a __tablename__ not a Class
  • it is possible to use a if in for loop in Jinja

Little reminder:

In Puppy owner = relationship('Owner', backref = 'pet', uselist=False)

In Owner, make sure you have the table name for the related table pup_id = db.Column(db.Integer, db.ForeignKey('puppies.id'))
