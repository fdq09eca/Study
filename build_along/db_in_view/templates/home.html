Welcome

This a mini-project. It builds a website that stores and reports data from a local SQLite datadase to the users. The goal of this project is getting myself familiar with the following technologies:

  • Flask
  • SQLAlchemy
  • Bootstrap
  • Jinja

Lesson I learned:

  • TYPE CAREFULLY. Otherwise you are spending 90% of time in finding bugs instead of being productive.
  • Make sure you are using the updated bootstrap version, when you are using 3rd party autocomplete package
  • READ CAREFULLY. The documentation of bootstrap is complete and do not skip line when you read it.
Here is the course for instruction, it is hard but rewarding, I learned a lot. Bootstrap is a very nice tool that saves me from typing hell lot of CSS like in Jekyll which demands a good understanding of the particular theme's CSS for making any adjustment.
